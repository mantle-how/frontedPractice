物件 Object 用來封裝其他資料的容器

物件的使用流程 :

建立物件 -> 使用物件

建立物件:
從一個空白的物件開始
new Object()

let obj = new Object();

建立物件成員 : (把資料放進物件裡面)
物件.成員名稱 = 資料

程式碼範例 :
let obj = Object();
obj.x = 3;
obj.y = 4;

函式也可以是物件中的成員

let obj = new Object();
obj.x = 3;
obj.y = 4;
obj.show = function(){
    console.log("Hello");
};


物件成員的種類:
物件的成員分成屬性以及方法兩中稱呼

let obj = new Object();
obj.x = 3; //成員非函式 稱為屬性
obj.y = 4; //成員非函式 稱為屬性
obj.show = function(){  //成員是函式 ， 成為方法
    console.log("Hello");
};


使用物件:
可以透過.存取物件的成員
物件.成員名稱

程式碼範例:

let obj = new Object();
obj.x = 3;
obj.y = 4;
obj.show = function(){
    console.log("Hello");
};
console.log(obj); // 他會把整個物件都印出來
console.log(obj.x + obj.y);
obj.show();

使用 this 關鍵字 : (在物件的方法中使用this代表物件本身)

let obj = new Object();
obj.x = 3;
obj.y = 4;
obj.show = function(){
    console.log(this.x , this.y);  //this.x = 3 ,this.y = 4
};

陣列物件 Array : 按照順序存放其他資料的容器
JavaScript 內建的物件 利用有順序的編號 來管理內部的資料

建立陣列物件:
建立空白的陣列 可放進變數中做後續操作

=> new Array  or  []

程式碼範例:
let arr = [];

在陣列中新增資料:
使用陣列物件的push方法:
陣列.push(資料)

程式碼範例:
let arr = [];
arr.push(3);
arr.push(4);


取得資料:
陣列中的每個資料都有一個編號

根據資料的編號取得資料: 陣列[編號]

程式碼範例 :
let arr = [];
arr.push(3);
arr.push(4)
console.log(arr[0]); // 3
console.log(arr[1]); // 4



取得陣列長度:
陣列長度代表陣列中的資料數量
利用 陣列的length屬令取得陣列長度

程式碼範例 :

let arr = [];
console.log(arr.length); // 0
arr.push(3);
arr.push(4)
console.log(arr.length); // 2

建立物件的同時 ， 同時建立初始資料
直接將資料放進[]之中 ， 用逗號隔開
[資料 , 資料　,...]

程式碼範例:
let arr = [3,4];
arr.push(0);


HTML DOM:

根據 HTML 標籤， 建立物件結構

每個 HTML 標籤在JS 引擎中都有對應的標籤物件

把 HTML 標籤物件， 串接在一起， 成為物件結構 就是HTML DOM

全名是 (Document Object Model)


瀏覽器如何管立 繪製html 網頁 :
html -> html dom in js -> 網頁畫面



操作window物件 : window物件是html dom 的入口

隨時可以用內建的window變數取得 console.log(window)


操作window物件 : 內建的screen 物件記錄使用者的螢幕資訊

screen 物件是window物件的一個屬性 :  windows.screen



操作document物件: 內建的document物件代表網頁主畫面

document物件是window物件的一個屬性 : window.document 可簡寫成document
