css基本語法 -> 屬性名稱.屬性內容

網頁一般預設文字 大約是16px

同時設定不同設定 中間加 ; 隔開

透過標籤的style 來套用 css 樣式

<h3 style = "Css 樣式"></h3>


乾淨的標籤 => 沒有預設css 樣式設定的標籤

有預設樣式的標籤
<h3> </h3> # 文字變粗體 預設就這樣

無預設樣式的標籤
<div> 就和一般文字一樣 </div>
<span>就和一般文字一樣 </span>


標籤的顯示模式

行內模式  inline  (都會在同一行)

<span> <a> <span>

區塊模式 block (自動換行)
<div> <h1>~ <h5> <ul> <table>

隱藏模式 : none  將標籤隱藏起來 => 需額外透過css 的display屬性設定




CSS Box Model 區塊模型 => 每一個標籤在畫面佔據一個方形區塊  像<div>就是區塊模式的標籤

區塊填塞 padding => 區塊的內容與邊框之間的距離

區塊邊框 Border => 區塊邊線寬度 樣式 及顏色

外邊距 Margin => 相鄰區塊之間的距離

區塊的寬度與高度 => 利用width 和 height 設定區塊尺寸


區塊的寬度:

    預設和父標籤等寬

    width:500px

    width:50%

區塊的高度:

    預設隨著的內容縮放

    height:300px

    height:100%

使用限制:

區塊模式 (block) 標籤才能設定尺寸
